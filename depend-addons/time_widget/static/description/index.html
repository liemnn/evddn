Time Widget

Time Widget Module is designed to enhance user experience by allowing users to select and input time quickly and accurately within Odoo forms. Whether through quick clicks or manual typing, this widget streamlines the way time values are handled in your system.



Note: This widget only works in Char fields

Technical Prerequisite : Users are expected to have basic technical knowledge to utilize this module.



Users can use these widgets for the char field widget = "flatpickr_time".

The user will see the time duration as 00:00:00 in the selected Scheduled Time field by default.

Users can either enter the time duration manually or select the hours and minutes using the time picker widget.

The Scheduled Time field is displayed in the HH:MM:SS format.

Is this app compatible with Odoo Enterprise as well as Community?

Yes, the app works perfectly well with Odoo Enterprise as well as Odoo Community. Odoo Online (Cloud) does not allow installation of third-party apps and hence this app cannot be installed on Odoo Online.
